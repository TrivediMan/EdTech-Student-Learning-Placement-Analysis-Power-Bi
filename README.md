# 🎓 EdTech Student Learning & Placement Analysis Dashboard

## 📌 Project Overview

The **EdTech Student Learning & Placement Analysis Dashboard** is a comprehensive Power BI solution designed to analyze student enrollment, attendance, placement performance, dropout trends, revenue generation, and learning modes across multiple regions and courses.

This dashboard helps educational institutions identify key performance indicators, improve student outcomes, optimize placement strategies, and enhance revenue generation.

---

## 🚀 Dashboard Objectives

- Monitor student enrollment and engagement.
- Analyze attendance patterns across courses.
- Track placement performance by course and region.
- Identify dropout trends and risk factors.
- Evaluate revenue contribution from courses and regions.
- Compare Online, Offline, and Hybrid learning modes.
- Support data-driven academic and business decisions.

# 📚 Attendance Analysis

## Course Average Attendance

| Course | Attendance % |
|----------|-------------|
| Web Development | 64.54 |
| Cloud Computing | 64.47 |
| Python | 64.41 |
| Cyber Security | 64.40 |
| Data Science | 64.39 |
| AI/ML | 64.28 |

### Key Findings

- Average attendance remains around **64-65%**.
- Most students fall into the **60-80% attendance range**.
- Attendance above **80%** significantly improves placement opportunities.
- AI/ML shows the lowest average attendance.

### Recommendations

- Implement attendance alerts and reminders.
- Introduce gamification and interactive learning.
- Encourage participation through engagement programs.
